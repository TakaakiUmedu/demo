適当な$i, j \ (1 \le i, j \lt \varphi(n))$に対して、$x_i$ と $k_i$ の定義より、

\[m c_i = n k_i + x_i\tag{1}\] \[m c_j = n k_j + x_j\tag{2}\]

$m(c_i - c_j) = nk_i + x_i - nk_j - x_j$

$m(c_i - c_j) = n(k_i - k_j)$

$c_i - c_j = 0$
